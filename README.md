# Prajwal Singh — Editorial Engineering Portfolio

A premium, magazine-style editorial portfolio showcasing systems-focused projects, technical case studies, and engineering toolkit.

## Features
- **Editorial Design**: Custom typography hierarchy using serif/sans font pairing, structured newspaper layouts, and micro-animations.
- **Interactive Case Studies**: Detailed problem-approach-outcome writeups with quantified engineering metrics and retrospective architecture analyses.
- **Toolkit Index**: Structured categories highlighting languages, libraries, database & cloud systems, and computer science foundations.
- **Timezone-Aligned Context**: Custom clock displaying accurate location times.
- **Client-Side Persistence**: Functional local draft saving for brief inquiries.

## Tech Stack
- **Core**: React, TypeScript, Vite, Tailwind CSS, Framer Motion
- **Design Tokens**: Custom HSL system color themes with signal red accent rules
